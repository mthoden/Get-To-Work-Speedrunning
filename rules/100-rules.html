45/60 Rules

Generic

  • All runs require permanent uninterrupted video proof in which no significant portion of the screen is obstructed and game audio is audible at all times.
  • Timing must be done using the In-Game Timer (IGT)
  • Entering a save file created outside of the run is banned
  • Using cheats or debug features during a run is banned
  • Altering game files in any way is banned
  • Use of any third-party programs which directly affect the game are banned, with the exception of software that caps your framerate such as RivaTuner (This includes rebinding software)

Category Specific

  • Timing starts upon creation of a new game when the In-Game Timer begins
  • Timing ends when you have collected both hats, collected all 19 podcasts, accomplished the Billionaire ending and the In-Game Timer stops
  • You may not quit out during a run, all runs must be completed in one go. You can however pause as long as it isn't to an egregious degree
  • If a glitch occurs unintentionally and does not affect the speed of the run, it will be considered allowed; this is subject to change at moderators discretion based on the severity of the glitch
  • Bypassing walls or accessing areas out of bounds is not allowed
  • Grabbing onto physics objects to launch yourself is not allowed
  • Abusing game mechanics to ragdoll and skip significant portions of the game is not allowed. (i.e. jump pad ragdolling)
  • Wall climbing is allowed
  • Grabbing onto the sides of launchers to be sent in unintended directions is allowed

Framerate

  • Although not required, it is highly encouraged that frame rate always be shown to avoid recategorization of your run. Rivatuner / Steam Overlay or other similar software works fine for this. FPS can be capped in-game by using the "exclusive full screen" mode; however, your method of capping FPS does not matter
  • 45/60 FPS: Framerate must be displayed and stay within 45 and 60 FPS for the entire duration of the run
  • 60 FPS: Framerate must be displayed and stay exclusively at 60 FPS for the entire duration of the run
  • No FPS: No FPS Display Framerate must not intentionally go below 45 FPS for the entire duration of the run. This will be enforced on a case by case basis. If your run is suspected of using tech exclusive to low FPS categories (such as Any%) it may be recategorized
